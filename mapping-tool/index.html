Welcome — How to use this tool

This tool helps you prepare HR data for Workday migration. Work through each section in order. Hover the ⓘ icon on any column to see what the field means, where the data comes from, and validation rules.

📋 Step-by-step

  • Follow sections in sidebar order (top → bottom)
  • Click + Add row to enter each record
  • Hover ⓘ on a column for full field guidance
  • Export when ready — template or with data

⚠️ Critical rules

  • Reference data loads before workers
  • Dates → YYYY-MM-DD format only
  • Yes/No fields → exactly Yes or No
  • IDs must be unique with no spaces

🔗 How data links together

Each section references IDs from earlier sections. A Position needs an Organization_ID from Supervisory Orgs and a Job_Profile_ID from Job Profiles. Validation in the tool will flag mismatches. The ⓘ tooltip on each reference field shows exactly which earlier sheet it must match.

📤 Two export modes

Export template: blank Excel with all sheets and column headers — share with clients to fill in offline.

Export with data: exports everything entered so far — ready to hand to your consultant for EIB upload.
